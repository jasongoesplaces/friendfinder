# friendfinder

### How It Works

This is a friend pairing app, where users answer a few questions and are then matched with a friend based on their answers.


### How To Use

A link to this app can be found [here!](https://polar-sea-87546.herokuapp.com/)


### Technologies Used

* node.js
* express.js
* mySQL
